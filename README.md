# Öğrenci Performans Analiz Sistemi

Bu proje, Python ve NumPy kullanılarak geliştirilmiş basit bir öğrenci performans analiz uygulamasıdır.  
Kullanıcıların öğrenci eklemesine, not girmesine ve bu veriler üzerinden analiz yapmasına olanak sağlar.

---

## Özellikler

- Öğrenci ekleme  
- Not ekleme (0–100 arası kontrol)  
- Öğrenci ve not listeleme  
- Detaylı analiz:
  - Ortalama (mean)
  - En yüksek ve en düşük not
  - Standart sapma
- Geçti / kaldı durumu belirleme  
- En başarılı öğrenciyi bulma  
- Sınıf ortalaması hesaplama  
- Öğrenci silme  
- Verileri JSON dosyasında kalıcı olarak saklama  

---

## Kullanılan Teknolojiler

- Python  
- NumPy  
- JSON  

---

## Örnek Çıktı

Ali -> Ortalama: 75.0 | Max: 90 | Min: 60 | Std: 12.24 | Durum: Geçti  
Mehmet -> Ortalama: 45.0 | Max: 50 | Min: 40 | Std: 4.08 | Durum: Kaldı  

Sınıf Ortalaması: 60.0
